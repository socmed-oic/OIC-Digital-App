# OIC Marketing & Analytics Dashboard

A comprehensive digital dashboard designed for OIC (encompassing OIC Spa and OIC Gym) to track and analyze marketing efforts, public relations, content strategies, and multi-outlet performance.

## Features

This application includes four main modules:

1. **Ads & Optimization**: Track key performance indicators (KPIs) such as Impressions, Avg. CTR, CPA, and ROAS. Includes multi-platform performance charts and AI-driven budget and targeting suggestions.
2. **PR & Exposure**: Monitor PR Cost vs Earned Media Value (ROI) and sentiment analysis. Includes an exposure tracker for media and influencers.
3. **Content Planning**: View trend radars and balance between timeless (e.g., education, relaxation) and updated (e.g., trends, viral music) content strategies. Includes a content calendar highlights section.
4. **Multi-Outlet Report**: Track performance across 250+ outlets, view regional campaign performance, and identify top-performing branches (e.g., OIC Spa Sudirman, OIC Gym Senayan).

## Tech Stack

- **HTML5**: Semantic structure.
- **CSS3**: Custom styling with `Inter` for UI and `Marcellus` for headings.
- **JavaScript**: App logic and interactivity.
- **Chart.js**: Dynamic data visualization.
- **FontAwesome**: Scalable vector icons.

## Getting Started

To run this project locally, simply clone the repository and open the `index.html` file in your preferred web browser. 

```bash
git clone https://github.com/socmed-oic/OIC-Digital-App.git
```

Open `index.html` in your browser. Alternatively, if you use a code editor like VS Code, you can use the Live Server extension.

## Deployment

The project includes a `vercel.json` configuration file, making it ready for instant deployment on [Vercel](https://vercel.com/).
